# Crossing Game

## Description

This project is a browser-based crossing game developed using HTML, CSS, and JavaScript. The objective of the game is to help a character cross a busy road while avoiding moving obstacles.

The gameplay is inspired by classic arcade-style crossing games where players must dodge traffic and survive as long as possible. These games are known for their simple mechanics and fast-paced reflex-based gameplay.

## Demo

Live Link:

## Features

* Character movement across lanes
* Moving obstacles (vehicles or objects)
* Collision detection system
* Increasing difficulty with progress
* Smooth and responsive controls
* Simple and clean UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## How to Run

1. Clone the repository:

```
git clone https://github.com/pranav2020-pixel/crossing-game.git
```

2. Open the project folder
3. Run `index.html` in a web browser

## Controls

* Arrow Keys to move the character

  * Up → Move Forward
  * Down → Move Backward
  * Left → Move Left
  * Right → Move Right

## Game Rules

* Cross the road while avoiding obstacles
* Each successful move increases progress
* Game ends when the player collides with an obstacle
